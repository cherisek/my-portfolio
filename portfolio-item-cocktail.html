Cocktail Website:Drink Recipes

make your favorite drinks

Cocktail website was a fun website I did where the user can look up different drink recipes and make some fun drinks at home.

For this website, I used TheCocktailDB API, React, React Hooks, Javascript, HTML, and CSS. I worked on the project individually.

I used React because ease of use in set up. As well as React, makes it easy to reuse components and be able to make use of the virtual DOM for faster reflection of updates.
